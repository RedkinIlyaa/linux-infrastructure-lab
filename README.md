
# Linux Infrastructure Lab

Educational Linux administration project built on Ubuntu 24.04.

## Goal

Prepare a Linux server for a small software development team and practice core Linux administration tasks.

## Topics

* Users and Groups
* Permissions
* Process Management
* Package Management
* Security
* Cron
* Bash Scripting
