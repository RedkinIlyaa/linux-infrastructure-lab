

## Goal
Learn how to inspect, monitor and terminate Linux processes.

## Process Discovery
Running processes were inspected using Linux monitoring tools.

A calculator application was started and its process was identified by name and PID.

## Process Monitoring
System resource usage and active processes were monitored before and after launching the calculator application.

## Process Termination
The calculator process was located and successfully terminated using its PID.

After termination, the process was no longer present in the process list.

## Used:
- top
- htop
- kill
